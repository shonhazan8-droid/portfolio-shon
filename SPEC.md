# Portfolio — Calibration Notes

**Direction:** Calm (calibrated to Cursor.sh)
**Font:** Geist 400/500/600 (neo-grotesque, matches Cursor) + JetBrains Mono for labels
**Accent:** Deep blue #1A3A6B — links only, never large fills
**Bg:** Warm off-white #F7F6F2 — not pure white, not cold gray

## Weight rules (DO NOT DRIFT)
- Headings: 500 (medium) — never 700/800
- Body: 400 regular
- Hierarchy via color + spacing + size, not weight jumps

## Layout
- Single column — everything stacks top to bottom
- Case studies: screenshot full width → text below (no side-by-side)
- About: portrait → lead text → body → stats (no side-by-side)
- Testimonials: dense grid 1→2→3 cols, small text, hairline top border

## DO NOT
- No bold/heavy weights
- No giant footer wordmark
- No marquee, no bracket mono labels [01 — ]
- No gradients, glow, glassmorphism
- No cold grays or pure white bg
- Accent on links only — not buttons, not fills
